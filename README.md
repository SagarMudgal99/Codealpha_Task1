# 🌸 Iris Flower Classification

This project implements a Machine Learning model to classify Iris flowers into three species: **Setosa, Versicolor, and Virginica** based on their sepal and petal measurements.

The project uses the famous Iris dataset provided by Scikit-learn and demonstrates the basic workflow of supervised machine learning, including data loading, model training, prediction, and performance evaluation.

## 🚀 Features

* Load the Iris dataset.
* Train a classification model.
* Predict flower species.
* Evaluate model accuracy and performance.
* Understand basic classification concepts in Machine Learning.

## 🛠 Technologies Used

* Python
* Scikit-learn
* NumPy
* Pandas

## ▶️ How to Run

```bash
pip install -r requirements.txt
python iris_classification.py
```

## 📊 Dataset

The Iris dataset contains 150 samples of Iris flowers belonging to three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

Each sample includes:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## 🎯 Objective

Build and evaluate a Machine Learning model capable of accurately classifying Iris flower species based on their measurements.
